# samOS

A nostalgic macOS Aqua-style web interface built with React and Tailwind CSS.

## Features

- Classic macOS Aqua design with pinstripes and gradients
- Interactive dock with hover effects
- Window management system
- Menu bar with dropdown menus
- Multiple app windows (Finder, Chat, iPod)
- Desktop icons
- Volume control
- Real-time clock

## Getting Started

### Install Dependencies

```bash
npm install
```

### Run Development Server

```bash
npm run dev
```

### Build for Production

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```

## Technologies

- React 18
- Vite
- Tailwind CSS
- Lucide React (icons)
